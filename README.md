# The Matrix 2.0

A highly accurate, browser-based recreation of the classic "Matrix" digital rain effect using HTML5 Canvas and JavaScript.

## Features
* **Glowing Heads**: The leading character of each stream glows pure white/cyan.
* **Pure Green Trails**: Trails are pure green without white color bleeding.
* **Dynamic Speed**: Automatically calculates screen height to ensure drops take exactly 4 seconds from top to bottom.
* **Clustering & Voids**: Streams dynamically cluster together and create organic voids (empty spaces) for a cinematic, irregular flow.
* **Random Glitches**: Characters within the fading trails randomly flip and change.
* **Easter Eggs**: Keep an eye out for hidden words occasionally dropping!

## How to Use
Simply open `index.html` in any modern web browser. No servers, dependencies, or build tools required.
